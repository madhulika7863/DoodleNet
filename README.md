#DoodleNet
# Network-Based Collaborative Whiteboard (Mini Jamboard)

A simple real-time **collaborative drawing board** built using **Flask** and **Socket.IO**.  
Multiple users can draw together on a shared whiteboard over the same network â€” just like a mini version of Google Jamboard!

---

## Features
-  Real-time drawing sync across devices  
-  User join/leave notifications  
-  Color picker and brush thickness control  
-  Undo last stroke  
-  Clear board for everyone  
-  Save board as an image (PNG)  
-  Works on local network (Wi-Fi)  

---

## Tech Stack
- **Frontend:** HTML5 Canvas, CSS, JavaScript  
- **Backend:** Python Flask + Flask-SocketIO  
- **Protocol:** WebSockets (real-time communication)
